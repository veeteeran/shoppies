# Overview for The Shoppies:trophy:

## What is this project?

This webpage will allow users to search for movie titles and add them to a list of nominees for The Shoppies.

## How does it work?

Type the title of a movie into the search bar and the results box shows a list of movies that match the text in the search bar.
<img width="1190" alt="Screen Shot 2021-05-08 at 2 39 52 PM" src="https://user-images.githubusercontent.com/14856004/117551753-501be180-b00d-11eb-9678-be8438594322.png">

Users can add titles to the nominees list by clicking the nominate button
<img width="1174" alt="Screen Shot 2021-05-08 at 2 40 28 PM" src="https://user-images.githubusercontent.com/14856004/117551759-56aa5900-b00d-11eb-8968-62706df13d7d.png">

Or remove from the nominees list clicking the remove button.
<img width="1175" alt="Screen Shot 2021-05-08 at 2 41 33 PM" src="https://user-images.githubusercontent.com/14856004/117551762-59a54980-b00d-11eb-9966-9d23f4cefd97.png">

Once five nominees are chosen a message appears alert the user of their choices.
<img width="1176" alt="Screen Shot 2021-05-08 at 2 42 05 PM" src="https://user-images.githubusercontent.com/14856004/117551767-5f029400-b00d-11eb-9f31-46536e29b7ae.png">
